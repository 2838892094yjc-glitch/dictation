# 学习历史记录功能 - 快速使用指南

## 功能简介

学习历史记录功能可以自动记录每次听写的成绩、用时、错题等信息，帮助你追踪学习进度和发现薄弱环节。

## 如何使用

### 1. 正常进行听写

按照正常流程进行听写：
1. 在"词库管理"页面选择单词
2. 点击"开始听写"
3. 完成听写
4. 在"答案批改"页面上传手写答案照片
5. 点击"开始识别并批改"

### 2. 自动保存记录

批改完成后，系统会自动保存本次听写记录，包括：
- 听写日期和时间
- 听写模式（英译中/中译英/拼写）
- 使用的词库名称
- 总单词数和正确数量
- 分数和用时
- 错误的单词列表

### 3. 查看历史记录

点击顶部导航栏的"📊 学习历史"进入历史页面，可以看到：

#### 学习统计
- 总听写次数
- 总单词数
- 平均正确率
- 总学习时长

#### 模式分布
- 柱状图显示各模式的使用次数
- 详细数据列表

#### 成绩趋势
- 折线图显示最近10次的分数变化
- 帮助你了解学习进步情况

#### 历史记录列表
- 按时间倒序显示所有听写记录
- 点击展开可查看详细信息
- 包括错题列表和用户答案

#### 高频错词
- 统计最容易出错的单词
- 显示每个单词的错误次数
- 帮助你针对性复习

### 4. 管理历史记录

#### 删除单条记录
在历史记录详情中点击"🗑️ 删除"按钮

#### 清空所有记录
点击"🗑️ 清空所有记录"按钮（需要二次确认）

#### 导出CSV
点击"📥 导出CSV"按钮，可以下载Excel格式的历史记录

## 使用技巧

### 1. 定期查看趋势
建议每周查看一次成绩趋势图，了解学习进度

### 2. 重点复习错词
关注"高频错词"列表，针对性地加强练习

### 3. 对比不同模式
查看模式分布，确保各种模式都有练习

### 4. 导出数据分析
定期导出CSV文件，可以用Excel做更深入的分析

### 5. 设定学习目标
根据平均正确率，设定提升目标

## 数据说明

### 分数计算
分数 = (正确数 / 总单词数) × 100%

### 评级标准
- 🏆 优秀：90分以上
- 👍 良好：80-89分
- ✓ 及格：60-79分
- ✗ 不及格：60分以下

### 用时统计
从点击"开始听写"到完成批改的总时间

## 常见问题

### Q: 历史记录保存在哪里？
A: 保存在 `data/history.json` 文件中

### Q: 如果不想保存某次记录怎么办？
A: 可以在历史页面中删除该记录

### Q: 历史记录会占用很多空间吗？
A: 不会，每条记录只有几KB，即使记录上千次也不会占用太多空间

### Q: 可以恢复已删除的记录吗？
A: 不可以，删除操作不可恢复，请谨慎操作

### Q: CSV文件用什么软件打开？
A: 可以用Excel、WPS表格等软件打开
