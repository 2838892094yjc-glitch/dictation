## 文件格式

### JSON 格式
```json
{
    "name": "我的词库",
    "words": [
        {"en": "apple", "cn": "苹果"},
        {"en": "banana", "cn": "香蕉"}
    ]
}
```

### TXT 格式
每行一个单词，中文用空格分隔：
```
apple 苹果
banana 香蕉
computer 电脑
```

## 实现步骤

### 步骤1: 创建导入导出函数
- import_from_json() JSON 导入
- import_from_txt() TXT 导入
- export_to_json() JSON 导出
- export_to_txt() TXT 导出

### 步骤2: 创建预置词库
- 收集 CET-4 常用词汇 (~2500词)
- 收集 CET-6 常用词汇 (~2500词)
- 收集高考词汇 (~3500词)
- 收集中考词汇 (~2000词)

### 步骤3: 添加 UI
- 词库管理页面添加导入按钮
- 词库管理页面添加导出按钮
- 预置词库选择列表

## 测试要点
1. 导入 JSON 正常
2. 导入 TXT 正常
3. 导出文件可再次导入
4. 预置词库加载正常
