# 快速开始指南

## 5分钟快速上手

### 步骤1: 启动应用 (1分钟)

```bash
cd /Users/yangjingchi/Desktop/自动听写
streamlit run app.py
```

浏览器自动打开：http://localhost:8501

---

### 步骤2: 添加单词 (2分钟)

#### 方法A: 手动输入（推荐新手）

1. 在"手动输入"框输入：
```
apple 苹果
banana 香蕉
computer 电脑
hello 你好
world 世界
```

2. 点击"➕ 添加到词库"
3. 点击"💾 保存词库"

#### 方法B: 拍照上传

1. 准备单词表照片
2. 点击"上传单词表照片"
3. 勾选"🤖 AI智能纠正拼写"
4. 上传照片，自动识别

---

### 步骤3: 开始听写 (2分钟)

1. **选择模式**（3选1）：
   - 英译中：听英文写中文 👈 推荐初学者
   - 中译英：听中文写英文
   - 拼写：听英文+中文拼写

2. **选择单词**：
   - 点击"全选"或勾选单词
   - 可选：勾选"🔀 打乱顺序播放"

3. **点击"🎧 开始听写"**

4. **听写流程**：
   - 点击"🔊 播放"听单词
   - 在输入框填写答案
   - 点击"✅ 确认答案"
   - 自动跳到下一个

5. **查看结果**：
   - 点击顶部"✅ 答案批改"
   - 查看正确率和详细结果

---

## 三种模式对比

| 模式 | 播报内容 | 填写内容 | 适合场景 |
|------|----------|----------|----------|
| **英译中** | 🔊 apple | ✏️ 苹果 | 测试中文释义记忆 |
| **中译英** | 🔊 苹果 | ✏️ apple | 测试英文拼写 |
| **拼写** | 🔊 apple, 苹果 | ✏️ apple | 拼写练习 |

## 进阶功能

### 快捷选择单词

- **全选**: 一键选择所有
- **前N个**: 选择前10个、20个等
- **字母范围**: 选择A-M、N-Z等

### 播放设置

- **英文音色**: 5种可选（默认：male_qn_qingse）
- **中文音色**: 5种可选（默认：female_shaonv）
- **播放间隔**: 1-10秒可调（默认3秒）

### 拍照批改

1. 在纸上按顺序写答案（每行一个）
2. 进入"答案批改"页面
3. 上传手写答案照片
4. 点击"🔍 开始识别并批改"
5. 查看详细批改结果

## 常见问题

### Q: 没有声音？
**A**: 检查系统音量，刷新页面重试

### Q: 拍照识别不准？
**A**: 确保光线充足，书写清晰

### Q: 如何切换词库？
**A**: 在词库页面使用下拉框选择

### Q: 数据会丢失吗？
**A**: 词库自动保存，不会丢失
